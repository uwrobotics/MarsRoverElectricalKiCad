## What is Design Rule Check(DRC)?

DRC is a validation process used to see if a PCB layout follows manufacturer and designer specified design guidelines.

Some key properity DRC will check over:
- Clearance: Physical Spacing between conduction elements
- Trace Width: Trace width satisfied the planned current load
